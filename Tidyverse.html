library(tidyr)
data2 <- gather(data1, 
                "t1", "t2", "t3",        # new key names (levels)
                key = "time",            # previous key name (factor name)
                value = "measure",       # to-assign values 
                factor_key = TRUE)       # saves as factor

A tibble data frame

  • it never creates row.names()

  • only vectors of length 1 are recycled

  • columns are built sequentially : they can make use of columns created in the same call

x = "a"
data.frame(x = 1, y = x)

## 1 1 a


x = "a"
tibble(x = 1, y = x)

##   <dbl> <dbl>
## 1     1     1

separate : separate each cell in a column to make several colums

data1

How to deal with NA values ?

complete : turn implicit missing values into explicit missing values.

data3
## # A tibble: 4 x 3
##   eleve  matiere   note
##   <chr>  <chr>    <dbl>
## 1 Thomas Maths       10
## 2 Thomas Francais    15
## 3 Marie  Maths       18
## 4 Tim    Francais     8
data3 = complete(data3, eleve, matiere)
data3
## # A tibble: 6 x 3
##   eleve  matiere   note
##   <chr>  <chr>    <dbl>
## 1 Marie  Francais    NA
## 2 Marie  Maths       18
## 3 Thomas Francais    15
## 4 Thomas Maths       10
## 5 Tim    Francais     8
## 6 Tim    Maths       NA

We can replace the NA values with the argument fill, alist that for each variable supplies a single value to use instead of NA.

data3
## # A tibble: 6 x 3
##   eleve  matiere   note
##   <chr>  <chr>    <dbl>
## 1 Marie  Francais    NA
## 2 Marie  Maths       18
## 3 Thomas Francais    15
## 4 Thomas Maths       10
## 5 Tim    Francais     8
## 6 Tim    Maths       NA
data3 = complete(data3, eleve, matiere, fill=list(note=0))
data3
## # A tibble: 6 x 3
##   eleve  matiere   note
